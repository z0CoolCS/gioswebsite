Let's get started

When we want to communicate with someone else we use words, and exactly a language. When we want to do the same with a computer machine, we also need a language. In this case, it is a programming language. Python is one of the most famous programming languages nowadays. If you learn to code in Python, you will be able to run your code in smartphones, tablets, laptops, IoT devices, etc. This is known as the multi-platform characteristic.

If you want to start to code, you need to understand an important concept which is Variable. We could think about a Variable as a Box which can save a specific value. This value belongs to a specific datatype.

Datatypes in Python

There exists several datatypes in python, but the most frequent are:
  • Numeric (Integer, Float, Complex)
  • Sequential (String, List, Tuple)
  • Dictionary
  • Set
  • Boolean

Numeric

Numeric datatypes take care of numeric values such as 4, 45.3 or 3+2j.

Integer
For an integer variable, we can define it using a integer value. The code below shows an example how you could do that üëá. 
variable1 = 23
variable2 = -2
variable3 = 0
Float
For an floating-point variable, we can define it using a floating-precision value. The code below shows an example how you could do that üëá. 
variable1 = 23.4
variable2 = -2.5
variable3 = 0.0
Complex
For an complex variable, we can define it using a complex value where the imaginary part has an extra character j. The code below shows an example how you could do that üëá. 
variable1 = 23+6j
variable2 = -2+2j
variable3 = 4j

Available operations using numerics values are: addition, substraction, multiplication, division, module. More info here 

variable1 = 2
variable2 = 5
variable3 = variable1 + variable2  # variable3 is 7
variable4 = variable1 - variable2  # variable3 is -3
variable5 = variable1 * variable2  # variable3 is 10
variable6 = variable1 / variable2  # variable3 is 0.4

String

For a String variable, we can define it using a value between characters " ". The code below shows some examples how you could do that üëá. 

variable1 = "23"
variable2 = "ABCDEFG"
variable3 = "ABC123abc"
List

For a list variable, we can define it using a sequential of values between [ ]. The code below shows some examples how you could do that üëá. 

variable1 = [2, 3, 5]
variable2 = ["ABC", "abc", "DEF"]
variable3 = [2, "ABC", 5.5]
Tuples

For a Tuple variable, we can define it using a sequential of values between ( ). The code below shows some examples how you could do that üëá. 

variable1 = (2, 3, 5)
variable2 = ("ABC", "abc", "DEF")
variable3 = (2, "ABC", 5.5)

Dictionary

Dictionary datatypes make easy to find a value based on a key. If you know about databases, dictionaries are quite similar to them. We can define it using a sequential pair of key-values between { }. The code below shows some examples how you could define a dictionary üëá. 

variable1 = {'a':'alphabet', 'b':'buy', 'c':'dinner'}
variable2 = {'a':2, 'b':'buy', 'c':[2, 3, 5]}
variable3 = {'a': {'ab':'ABC', 'ac':'ACD'}, 'b':'buy', 'c':[2, 3, 5]}

To access a specific value inside a dictionary we use the variable and a key. 

variable1 = {'a':'alphabet', 'b':'buy', 'c':'dinner'}
print(variable1['a']) # it shows 'alphabet'

Set

Set datatypes make easy to find a unique values in a group. We can define it using a sequential of values between { }. The code below shows some examples how you could define a set üëá. 

variable1 = {'a', 'b', 'c'}
variable2 = {'a', 2, 2.3}
variable3 = {"cat", "dog", "bird", "dog"}
print(variable3) # it will show {'dog', 'bird', 'cat'}

Note: one think to take in consideration is we can not have dictionaries or list inside a set datatype.

Boolean

Boolean datatypes takes only two values True or False. This is called a binary variable. The code below shows some examples how you could define a set üëá. 

variable1 = True
variable2 = False
variable3 = not variable1
print(variable3) # it will show False

We usually see boolean expressions when we compare values. The code below shows some examples how you could define a set üëá. 

variable1 = 20
variable2 = 13
variable3 = 20 
print(variable1 == variable2) # it will show False
print(variable1 == variable3) # it will show True
print(variable1 >= variable2) # it will show True
print(variable2 <= variable3) # it will show True
